# recipe_plan
# Recipe Planner / Рецептурний сайт з плануванням меню

A web application for managing recipes and creating personalized meal plans.
Вебдодаток для керування рецептами та створення персоналізованого меню.

---

##  Purpose / Мета

**English:**
The Recipe Planner allows users to:

* Browse and search recipes by name or category.
* View detailed recipe instructions with ingredients and cooking steps.
* Add, edit, and delete recipes.
* Plan daily or weekly menus.
* Organize recipes by categories (e.g., Breakfast, Lunch, Dinner, Desserts).
* Create a shopping list based on the planned menu.
* Optionally, mark favorite recipes for quick access.

**Українська:**
Рецептурний сайт дозволяє користувачам:

* Переглядати та шукати рецепти за назвою або категорією.
* Переглядати детальні інструкції приготування з інгредієнтами та кроками.
* Додавати, редагувати та видаляти рецепти.
* Планувати щоденне або тижневе меню.
* Організовувати рецепти за категоріями (Сніданок, Обід, Вечеря, Десерти).
* Створювати список покупок на основі запланованого меню.
* За бажанням позначати улюблені рецепти для швидкого доступу.

---

## Features / Функціонал

**English:**

* Recipe list with search and category filtering.
* Detailed recipe view with ingredients and preparation steps.
* Add, edit, and delete recipes.
* Menu planner for daily and weekly meals.
* Automatic shopping list generation.
* Favorites system for quick access to loved recipes.

**Українська:**

* Список рецептів із пошуком та фільтрацією за категоріями.
* Детальний перегляд рецепту з інгредієнтами та кроками приготування.
* Можливість додавання, редагування та видалення рецептів.
* Планувальник меню для щоденних і тижневих страв.
* Автоматичне створення списку покупок.
* Система улюблених рецептів для швидкого доступу.
